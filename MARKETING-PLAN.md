# WhatsWebsite — Paid Marketing Plan 🚀

> Goal: Acquire users via Click-to-WhatsApp (CTWA) ads + Google Search.
> Budget: Start $500/mo → Scale to $900/mo by Month 5.

---

## Phase 0: Setup (Before Spending)

### WhatsApp Business API
- [ ] Get API via BSP → **Interakt** (~$15/mo) or Wati (~$39/mo)
- [ ] Connect WhatsApp number → Facebook Business Manager → Business Settings → Accounts → WhatsApp Accounts
- [ ] Link Facebook Page to WhatsApp number

### Bot Flow (Build before ads go live)
```
Welcome message
  → Quick-reply buttons: Restaurant / Shop / Services / Other
  → Ask business name
  → Show website preview within 60 seconds
```

### Tracking
- [ ] Set up **Meta Conversions API (CAPI)** via BSP
- [ ] Track: conversation started → website created → premium purchase

---

## Phase 1: CTWA Campaign — Weeks 1–2
**Budget: $500/mo (100% Meta)**

### Campaign Setup (Ads Manager)
- Objective: **Engagement** → Conversion location: **Messaging Apps → WhatsApp**
- Optimize for: **Conversations**
- Enable **CBO** (Campaign Budget Optimization), set **$13–15/day**

### Ad Sets

| Ad Set | Targeting |
|--------|-----------|
| **#1 Interest** | Age 25–55 · Small Business Owners · Entrepreneurship · FB Page Admins · Restaurant Management · **Exclude:** Web Dev, Coding, WordPress |
| **#2 Broad** | Age 25–55 · Same location · No interests — Advantage+ finds converters |

### Ad Creatives (make 3–4 variations)

**Priority: Video** (15–30 sec vertical)
- Screen recording: person building a website via WhatsApp chat
- Add captions for silent viewers

**Carousel**
- Slide 1: Before (no website, paper menu)
- Slide 2: WhatsApp chat building the site
- Slide 3: After (finished website live)

**Testimonial**
- Quote from early user + screenshot of their website

### Ad Copy (PAS Framework)
```
Problem:   "Still don't have a website for your business?"
Agitate:   "Developers charge ₹50,000+. DIY builders are complicated."
Solution:  "Create one by chatting on WhatsApp. Free. 5 minutes."

CTA: "Chat Now →"
```

### Pre-filled Message + Icebreakers
```
Pre-fill: "Hi! I want to create a website for my business 🏪"

Icebreaker buttons:
  🍽️  I own a restaurant
  🛍️  I have a shop
  💇  I have a salon
  🔧  I offer services
```

> ⚠️ **Publish and DON'T TOUCH for 7 days** — learning phase.

## Phase 2: Optimize — Weeks 3–4

### Metrics to Check
| Metric | Target |
|--------|--------|
| Cost per conversation | $1–$3 |
| CTR | >1.5% |
| Conversation → Website rate | >25% |

### Actions
- Kill ads with cost >2x average after 1,000+ impressions
- Duplicate winners with slight copy/creative variations
- Launch **retargeting campaign** ($3–4/day, ABO):
  - Video viewers (50%+ watched)
  - Page engagers
  - People who clicked but didn't message

### 72-Hour Free Window Optimization
```
T+0:    Instant welcome → show template options
T+1–4:  Follow-up if dropped off ("Stuck? I can help 🙋")
T+24:   Send 2–3 website examples in their niche
T+48:   Final nudge before window expires ("Last chance to create free!")
```

## Phase 3: Add Google Ads — Month 3+
**Budget: $400 Meta / $100 Google**

### Google Search Campaign
- Bidding: Manual CPC to start → switch to **Maximize Conversions** after 15+ tracked
- Schedule: Business hours only, $3–5/day
- Send traffic to landing page with **big green WhatsApp button** (wa.me link)

### Keywords (Long-tail only — low CPC ₹8–25)
```
✅ "create website on WhatsApp"
✅ "build website from phone"
✅ "website builder no computer"
✅ "WhatsApp pe website kaise banaye"
✅ "free website for restaurant India"
```

### Negative Keywords
```
❌ Wix, Squarespace, WordPress, Shopify
❌ coding tutorial, developer job, web development course
❌ website builder (too broad, $5–15+ CPC)
```

### Tracking
- [ ] Google Tag Manager on landing page
- [ ] Track WhatsApp button clicks as conversions

## Phase 4: Scale — Month 5+

### Budget Split (at scale)
| Channel | % |
|---------|---|
| Meta Prospecting | 50% |
| Meta Retargeting | 15% |
| Google Search | 25% |
| Testing (new creatives/audiences) | 10% |

### Lookalike Audiences
1. Upload WhatsApp contacts who created websites → **1% Lookalike**
2. New ad set under existing CBO campaign
3. Add 2–3% lookalike as separate ad set for comparison

### North Star Metric
> Always optimize toward **cost per website created**, not cost per conversation.
